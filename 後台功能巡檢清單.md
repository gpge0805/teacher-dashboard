# 後台功能巡檢清單

## 巡檢目的

確認 teacher-dashboard 在加入新題庫後，登入、權限、成績查詢、每週統計、錯題分析與主檔管理功能仍可正常使用，且新檢定別不會被當成未標記或遺漏。

## 本次重點檢定別

- industrial：工丙
- digital-b：數乙
- industrial-wiring-c：工配丙
- computer-hardware-b：硬裝乙
- computer-hardware-c：硬裝丙

## 巡檢前準備

- [ ] Supabase 的 exam_results 已存在至少 1 筆舊檢定資料與 1 筆新檢定資料。
- [ ] 新檢定資料的 exam_settings.certificationId 有正確寫入。
- [ ] 如需驗證 SQL 補值，先確認 [sql/add_certification_id_to_exam_results.sql](sql/add_certification_id_to_exam_results.sql) 已執行或準備執行。
- [ ] 準備 2 種帳號：admin 1 組、teacher 1 組。
- [ ] teacher 帳號至少綁定 1 個班級與 1 位學生。

## 1. 登入與權限

### 1-1 登入畫面

- [ ] 輸入正確帳密可成功登入。
- [ ] 輸入錯誤帳密會顯示錯誤訊息，不可登入。
- [ ] 停用帳號不可登入。

### 1-2 權限導覽

- [ ] admin 可看到教師帳號管理。
- [ ] teacher 看不到教師帳號管理。
- [ ] 登出後 session 會清除，重新回到登入頁。

## 2. 成績報表查詢

### 2-1 基本資料載入

- [ ] 可正常載入 exam_results 資料。
- [ ] teacher 只看到自己學生資料。
- [ ] admin 可看到全部學生資料。

### 2-2 篩選功能

- [ ] 班級篩選正常。
- [ ] 姓名或學號搜尋正常。
- [ ] 日期區間篩選正常。
- [ ] 工作項目篩選正常。

### 2-3 檢定別篩選

- [ ] 檢定別下拉可看到：工丙、數乙、工配丙、硬裝乙、硬裝丙、未標記。
- [ ] 各檢定別切換後，資料筆數會跟著變化。
- [ ] 新檢定資料不會被錯誤歸到未標記。
- [ ] certification_display 顯示正確中文名稱。

### 2-4 資料清理功能

- [ ] 無效資料清理清單可正常列出。
- [ ] 重複成績清理可正常判斷、預覽與刪除。
- [ ] 刪除動作需要二次確認。

### 2-5 匯出與列印

- [ ] 篩選後資料可正常匯出 CSV。
- [ ] 列印版面可正常開啟。

## 3. 每週成績統計

### 3-1 週期設定

- [ ] 可正常讀取全局週期設定。
- [ ] 可正常儲存全局週期設定。
- [ ] 可正常讀取班級及格標準。
- [ ] 可正常儲存班級及格標準。

### 3-2 統計結果

- [ ] 指定班級後可正確產出本週統計表。
- [ ] 無成績時會顯示提示，不會噴錯。
- [ ] 關鍵時段與其他 4 筆高分計算結果合理。

### 3-3 檢定別篩選

- [ ] 檢定別下拉可看到：工丙、數乙、工配丙、硬裝乙、硬裝丙、未標記。
- [ ] 選取新檢定別後，統計表只剩該檢定資料。
- [ ] 若當週無該檢定資料，會顯示當週沒有成績。

## 4. 錯題弱點分析

### 4-1 題庫載入

- [ ] 可正常載入各檢定別題庫。
- [ ] 電腦硬體乙/丙級會一併吃到 questions-shared-info.json 的共用科目。
- [ ] 沒有題庫檔時，會顯示明確錯誤訊息。

### 4-2 分析結果

- [ ] 工作項目總覽可正常顯示總題數、作答人次、錯題數、總錯誤次數、錯誤率。
- [ ] 展開逐題分析後，可對到完整題目與正確答案。
- [ ] 匯出工作項目 CSV 正常。
- [ ] 匯出完整錯題分析 CSV 正常。

### 4-3 新檢定別驗證

- [ ] 新檢定別資料可以被正確解析，不會全部落到未標記。
- [ ] 若同一篩選範圍內含多個檢定別，相同工作項目名稱不會互相混算。
- [ ] 顯示名稱會帶出檢定別前綴，方便辨識不同題庫來源。

## 5. 學生名冊管理

### 5-1 目前名單

- [ ] teacher 只看得到自己學生。
- [ ] admin 可看全部學生。
- [ ] 班級篩選與學號/姓名搜尋正常。
- [ ] 勾選後可批次刪除學生。

### 5-2 單筆新增 / 修改

- [ ] 同學號 upsert 正常，不會新增重複資料。
- [ ] 必填欄位未填會擋下。

### 5-3 批次匯入

- [ ] CSV 匯入正常。
- [ ] Excel 匯入正常。
- [ ] 缺少必要欄位時會提示錯誤。
- [ ] 匯入後學生資料可正確新增或更新。

## 6. 教師帳號管理

### 6-1 權限限制

- [ ] 只有 admin 可進入本頁。
- [ ] teacher 強行進頁時會被擋下。

### 6-2 教師名單與設定

- [ ] 可直接編輯姓名、權限、啟用狀態。
- [ ] 儲存後重新整理仍保留。

### 6-3 新增教師

- [ ] 可新增 teacher 帳號。
- [ ] 可新增 admin 帳號。
- [ ] 重複帳號會擋下。
- [ ] 新增後可實際登入。

## 7. 公開學生每週查詢

- [ ] 使用 ?view=student-weekly 可進入公開查詢頁。
- [ ] 輸入存在學號可查到資料。
- [ ] 不存在學號會顯示錯誤訊息。
- [ ] 週次切換正常。
- [ ] 成績組成、及格標準、結果顯示正確。

## 8. 新檢定別資料庫巡檢

- [ ] exam_results.certification_id 已存在。
- [ ] 新資料上傳後 certification_id 會正確寫入。
- [ ] 若前端漏傳 certification_id，但 exam_settings.certificationId 有值，trigger 可自動補上。
- [ ] 舊資料若 exam_settings.certificationId 有值，可回填成功。
- [ ] categories 含工作項目10的舊資料仍可回填為 digital-b。

## 9. 建議抽測樣本

- [ ] 每個檢定別至少抽 1 筆 exam_results 測成績報表。
- [ ] 每個檢定別至少抽 1 筆含 wrong_question_ids 的資料測弱點分析。
- [ ] 至少抽 1 個班級測每週統計。
- [ ] 至少抽 1 位學生測公開查詢頁。
